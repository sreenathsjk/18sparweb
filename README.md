## Getting Started

### Run Locally

```bash
# Clone the repository
git clone https://github.com/sreenathsjk/18spar.git

# Navigate into the folder
cd 18spar

# Open in browser (no build step needed)
open index.html
```

### Deploy to Vercel

```bash
# Install Vercel CLI
npm install -g vercel

# Deploy
vercel
```

Or simply drag and drop the project folder at [vercel.com/new](https://vercel.com/new).
